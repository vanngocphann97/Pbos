# PBOS v3.1 Workflow Engine

## Daily Workflow

1. Capture: mọi ý tưởng, việc, link, thông tin đưa vào Inbox.
2. Decide: phân loại sang Project, Learning, Content, Portfolio hoặc Archive.
3. Execute: chọn tối đa 3 Focus mỗi ngày.
4. Review: cuối tuần trả lời 4 câu hỏi bắt buộc.
5. Convert: output tốt chuyển thành Portfolio Asset.

## Non-negotiable Rules

- Không nhập thẳng vào Project nếu chưa rõ lý do.
- Không đưa quá 3 Focus vào Today.
- Không kết thúc tuần nếu chưa Review.
- Không để output tốt nằm chết trong folder.
- Không đưa dữ liệu mật lên GitHub Public.

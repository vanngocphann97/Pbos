# PBOS v1.3 Operating Guide

## Mục tiêu

PBOS là website cá nhân độc lập, dữ liệu nằm trong một file duy nhất: `data/pbos-data.json`.

## Cập nhật hằng ngày

Sửa nhóm:

```json
"today"
```

Các phần cần sửa:

- focus
- currentFocus
- mainRisk
- nextOutput

## Cập nhật hằng tuần

Sửa:

- kpi
- projects
- learning
- content
- portfolio

## Cập nhật thành tích

Sửa:

```json
"career": {
  "capital": [],
  "achievements": []
}
```

## Không đưa dữ liệu mật lên GitHub Public

Không upload:
- Quyết định khen thưởng bản scan nếu có thông tin nội bộ
- Hợp đồng
- CCCD
- Hồ sơ tài chính
- Tài liệu công ty chưa được phép công khai

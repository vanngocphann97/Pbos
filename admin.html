Quy trình cập nhật

  1. Bấm Load data.
  2. Sửa dữ liệu trong ô JSON bên dưới.
  3. Bấm Export JSON để tải file pbos-data.json.
  4. Upload file đó lên GitHub tại data/pbos-data.json.
  5. Commit changes và mở lại website.
